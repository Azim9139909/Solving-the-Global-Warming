## THE PLAN
> Вам будет предложен небольшой тест о климате, состоящий примерно из 10 вопросов. После прохождения теста вы получите оценку по 10-балльной шкале, указывающую на ваши результаты. Внизу будет указана тема, на которую вы ответили неправильно.
